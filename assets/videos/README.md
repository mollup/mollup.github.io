# Videos Directory

Place all video assets here.

## Required Videos:

### 1. `matchpoint-demo.mp4`
- **Duration:** ~2 minutes
- **Resolution:** 1080p (1920x1080) or 720p (1280x720)
- **Format:** MP4 (H.264 codec)
- **Content:** Platform walkthrough showing:
  - Tournament creation
  - Bracket management
  - Live scoring
  - Results tracking
- **Requirements:**
  - ✅ Must have voiceover narration
  - ✅ Must have captions (via VTT file)
  - ✅ Font sizes must be magnified for legibility
  - ✅ Show all key features

### 2. `matchpoint-demo-captions.vtt`
- **Format:** WebVTT
- **Content:** Time-synced captions for entire video
- **Example:**
  ```
  WEBVTT

  00:00:00.000 --> 00:00:03.000
  Welcome to MatchPoint, the ultimate tournament platform.

  00:00:03.000 --> 00:00:07.000
  Let's see how easy it is to create your first tournament.
  ```

## Video Creation Tips:
- Use screen recording software: OBS Studio, QuickTime, Loom, ScreenFlow
- Record at highest quality possible
- Use clear, enthusiastic narration
- Add professional captions (YouTube auto-captions + manual editing, or Rev.com)
- Compress video for web without losing too much quality
- Test playback in multiple browsers
